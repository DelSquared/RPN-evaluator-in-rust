# RPN-evaluator-in-rust
Reverse Polish notation evaluator in Rust (stack based example)

## Output

```bash
$> rustc -o main main.rs
$> ./main
[]
tkn = "2"
[2]
tkn = "2"
[2, 2]
tkn = "+"
[4]
tkn = "1"
[4, 1]
tkn = "-"
[3]
result = [3] (quick maths)
$>
```
